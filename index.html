Project Overview

VimMaster is an open-source, browser-based Vim learning platform built with vanilla JavaScript. The codebase is designed for extensibility and educational purposes.

  • ‚úÖ Vanilla JavaScript - No frameworks, easy to understand and modify
  • ‚úÖ Modular Architecture - Clean separation of game logic and UI
  • ‚úÖ Extensible API - Add custom levels and commands easily
  • ‚úÖ Educational Focus - Well-documented code for learning

Quick Setup

git clone https://github.com/renzorlive/vimmaster.git

cd vimmaster

# Serve locally

python -m http.server 8000

# or

npx serve .

No build process required. Pure HTML/CSS/JS that runs in any modern browser.
